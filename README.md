# 🔍 AI-Text-Detector: Python AI Detector

A high-performance AI text detection tool built with Python, using a hybrid approach of **Transformer-based classification** and **Linguistic Perplexity analysis**.

## 🚀 Features
* **Hybrid Detection:** Combines RoBERTa-base classification with GPT-2 perplexity metrics.
* **Real-time Analysis:** Get probability scores in seconds.
* **User-Friendly UI:** Built with Streamlit for a clean, interactive experience.
* **2026 Ready:** Optimized for detecting modern LLM outputs (GPT-4, Claude 3.5, etc.).

## 🛠️ Tech Stack
* **Language:** Python 3.10+
* **ML Frameworks:** PyTorch, Hugging Face Transformers
* **Dashboard:** Streamlit
* **Linguistics:** NLTK / GPT-2 Tokenization

## 📦 Installation
1. Clone the repo:
   `git clone https://github.com/ShivadityaAdam/Ai-Text-Detector.git`
2. Install dependencies:
   `pip install -r requirements.txt`
3. Run the app:
   `streamlit run app.py`
